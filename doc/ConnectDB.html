ConnectDB

Conception ››

Dans cette classe, nous réalisons :


    • La création d' une connexion à la base de données selon les informations de l'objet ConnectDB.
    • Cette connexion est affectée à la variable membre $dbConnection.
    • la fonction dbQuery($query) : Retourne le résultat de la requete sous la forme d'un tableau de tableaux associatif avec comme argument $query qui est une chaine de caractère représentant la requete à exécuter sur la base de données
    • la fonction dbCRUD($query) : Retourne le résultat de la fonction exec() cette méhode permet de gérer l'exception que peut générer exec() avec comme argument $query qui est une chaine de caractère représentant la requete à exécuter sur la base de données
